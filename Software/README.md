- [数据库](https://github.com/syntomic/WebApp/tree/master/database/)
    - RDBMS
        - 特点
            - 完备的数学理论基础
            - 完善的事务管理机制
            - 高效的查询处理引擎
        - 基石
            - ACID
            - SQL
    - NoSQL
        - 特点
            - 灵活的可扩展性
            - 灵活的数据模型
            - 与云计算紧密融合
        - 基石
            - CAP
            - BASE
            - 最终一致性
    - 实例
        - MySQL: 关系型数据库
        - MongoDB: 文档型数据库
        - Redis: 基于内存的键-值数据库
        - Neo4j: 图数据库
